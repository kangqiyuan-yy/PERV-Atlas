Data pipeline at a glance

From raw genomic scan to interactive in-browser exploration — every step is reproducible from the source files in this repository.

1

Identification

RetroTector / LTR-finder + manual curation locate 1,165 intact PERV proviruses across diverse pig breeds.

2

Classification

Phylogenetic analysis splits γ.ERV vs β.ERV, breed groups, and computes Identity / Kimura distance / insertion time.

3

Annotation

139 high-confidence sequences receive ORF (LTR / GAG / POL / ENV) and structural-domain annotations in BED format.

4

Visualization

FASTA / Excel / BED / GTF are exposed via a Flask backend and a custom igv.js front-end with rich interactivity.
